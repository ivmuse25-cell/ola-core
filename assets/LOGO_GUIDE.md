# Logo Usage Guide

## Logo Variations

OLA has two official logo variations:

### 1. Icon Logo (`assets/logo_icon.png`)
- **Use for**: App icons, favicons, GitHub avatar, social media profile pics
- **Format**: Square (1:1 ratio)
- **Minimum size**: 64x64px
- **Ideal sizes**: 512x512px (app icon), 180x180px (favicon)

### 2. Banner Logo (`assets/logo_banner.png`)  
- **Use for**: README headers, social media banners, presentations
- **Format**: Horizontal banner (wide)
- **Minimum width**: 600px
- **Ideal width**: 1200px

## Color Palette

Official OLA colors:

- **Deep Blue**: `#1E3A8A` - Security, trust, stability
- **Purple**: `#8B5CF6` - Identity, innovation
- **Violet**: `#6366F1` - Technology, modern
- **Amber**: `#F59E0B` - Authentication, warmth
- **Gray**: `#6B7280` - Text, subtle elements

## Design Concept

The logo combines two biometric concepts:
- **Eye**: Face recognition, vision, authentication
- **Fingerprint**: Identity, security, uniqueness

The fingerprint patterns within the iris symbolize:
- Multi-factor biometric fusion
- Privacy (hidden patterns)
- Security layers

## Usage Guidelines

### ✅ Do:
- Use official color palette
- Maintain aspect ratios
- Ensure minimum sizes for readability
- Use on light or dark backgrounds (logos have transparency)
- Crop with adequate padding

### ❌ Don't:
- Distort or stretch logos
- Change colors (use official palette)
- Add drop shadows or effects
- Use low-resolution versions
- Rotate or skew

## File Formats

- **PNG**: For web, GitHub, documentation (current)
- **SVG**: Vector format (coming soon - ask maintainer)
- **ICO**: Windows icons (can convert from PNG)
